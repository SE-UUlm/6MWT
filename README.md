# 6MWT 2026
The (6MWT) Six-Minute Walk Test is a medical test to quickly assess the functional physical performance of a patient. The patient has to walk 6 minutes and the distance traveled is measured. Based on age and body height, a fitness score is calculated.
There is a pre-existing app that uses step counting to perform the test.

The goal of our Software Engineering Project is to explore other options for measuring the distance traveled, e.g. GPS, smartwach sensors.

More information regarding the test: [6mwt.org](https://www.6mwt.org/)
